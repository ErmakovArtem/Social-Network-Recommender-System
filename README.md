# Social-Network-Recommender-System
## Описание проекта
Это проект по построению рекомендательной системы: от выгрузки данных из PostgreSQL до развёртывания сервиса, который возвращает персонализированные рекомендации (список релевантных постов для пользователя).

В основе решения — машинное обучение: представлено формирование фичей, обучение модели, и интеграция её в сервис, который развёртывается через uvicorn.
